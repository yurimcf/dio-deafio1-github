
# Códigos Iniciais do Git

git clone <https://link-com-o-nome-do-repositório>
git push -u <local-remoto> <nome-da-branch>
git status
git add <arquivo> 					---- Adiciona somente um arquivo 
git add -A 						---- Adiciona todos os aruivos
git commit -m "mensagem do commit"
git push <repositório-remoto> <nome-da-branch> 		---- Envia para o repositório 
git push -u origin <nome-da-branch>
git pull <repositório-remoto>				---- Obtem atualizzações do repositório
